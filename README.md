# C для удовольствия
Здесь живут маленькие игровые проекты на C, написанные в свободное время — просто потому что программировать интересно.
Проекты работают прямо в консоли linux, используют минимальное количество системных вызовов и не требуют графических библиотек.

> "Код, рожденный в свободное время из чистого интереса и желания творить"
